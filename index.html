2. TN·TT·IT 계통 — 회로도와 KEC 판정식

TT 방식

변압기 N──⏚ RB
　L ───── 부하 외함
　　　　　│ PE
　　　　　⏚ RA
귀로: 외함 → RA → 대지 → RB → N

대지를 포함한 귀로 때문에 고장전류가 작아질 수 있어 RCD·ELB가 핵심입니다.

RA × IΔn ≤ UL

예: 30mA RCD, 허용접촉전압 50V라면 이론상 RA ≤ 50÷0.03 = 1,667Ω. 단, 실제 현장은 신뢰성·서지·계절변동을 고려해 훨씬 낮게 관리합니다.

TN 방식

변압기 N·PE 결합점 ⏚
　L ───── 부하 외함
　　　　　│ PE ─────┘
귀로: 외함 → PE/PEN → 변압기 중성점

금속성 보호도체로 저임피던스 귀로를 구성하여 과전류차단기의 신속 동작을 기대합니다.

Zs × Ia ≤ U0

예: U0=220V, 차단에 필요한 전류 Ia=100A라면 Zs ≤ 2.2Ω.

IT 방식

전원측 비접지 또는 고임피던스 접지
　L ───── 부하 외함
　　　　　│ PE
　　　　　⏚
1차 지락전류가 작아 운전 지속 가능

첫 지락 후 즉시 정전하지 않을 수 있지만 절연감시장치(IMD), 경보, 신속한 고장점 탐지가 필수입니다.

Id × RA ≤ UL

판정 기준: 접지극 개수가 아니라 외함 고장전류가 어떤 도체와 경로로 전원점까지 돌아가는지, 그리고 N·PE가 어디에서 연결되는지를 봅니다.

전류전극 C로 시험전류를 흘리고, E–P 사이 전압을 측정하여 접지저항을 계산합니다. P 위치를 앞뒤로 이동했을 때 값이 안정적인지도 확인합니다.

2단자법 해석

R표시 ≈ R피측정 + R보조 + R접속

공통접지망을 분리하지 않으면 여러 접지극이 병렬이 되어 낮게 표시될 수 있습니다.

Req = 1 ÷ (1/R₁ + 1/R₂ + ··· + 1/Rₙ)

예: 10Ω 접지극 2개가 이상적으로 병렬이면 합성은 5Ω입니다. 따라서 MCC반에서 측정한 낮은 값이 개별 접지극 자체의 값이라고 단정하면 안 됩니다.

5. 절연저항 측정 — 원리, 판정과 안전 절차
Rins = Vtest ÷ Ileak

메거는 직류 시험전압을 인가하고 흐르는 미소 누설전류로 절연저항을 계산합니다. 값이 클수록 일반적으로 절연상태가 좋습니다.

1. 정전·검전
2. 잔류전하 방전
3. PLC·인버터·SPD 분리 검토
4. 시험전압 선정
5. 측정값·시간 기록
6. 재방전·원상복구

계산 예: 500V 시험에서 누설전류가 0.5mA라면 R = 500÷0.0005 = 1MΩ.

주의: 전자회로, 인버터, PLC, 계측기, SPD·MOV가 연결된 상태에서 무조건 DC 500V를 인가하면 손상될 수 있습니다. 제조사 지침에 따라 분리 또는 시험전압 저감을 검토합니다.

10. 진상용 콘덴서 — 역률개선 계산과 점검
Qc = P × (tanφ₁ − tanφ₂)

P는 유효전력(kW), φ₁은 개선 전 역률각, φ₂는 목표 역률각이며 Qc는 필요한 콘덴서 용량(kvar)입니다.

예: 100kW 부하의 역률을 0.75에서 0.95로 개선할 때
tanφ₁≈0.882, tanφ₂≈0.329 → Qc=100×(0.882−0.329)=55.3kvar. 실제 선정은 표준 단계와 부하변동을 고려합니다.

Ic = Qc ÷ (√3 × VLL)

380V, 50kvar: Ic≈50,000÷(1.732×380)=76A.

  • 부하 또는 모선에 병렬 설치하고 자동역률조정기로 단계 제어합니다.
  • 과보상 시 진상역률·전압상승이 발생할 수 있습니다.
  • 고조파가 많은 계통은 직렬리액터와 공진 검토가 필요합니다.
  • 정전 후 방전시간과 잔류전압을 확인한 뒤 작업합니다.
  • 팽창·누유·이상음·단자과열·전류불평형·퓨즈단선을 점검합니다.
