      <details open><summary>1. 접지의 목적과 감전보호 원리</summary><div class="training-body">
        <h3 class="training-title">■ 접지는 무엇을 보호하는가</h3>
        <ul>
          <li><b>고장보호:</b> 절연파괴로 노출도전성부분에 전압이 나타날 때 보호도체(PE)를 통해 고장전류의 귀로를 확보합니다.</li>
          <li><b>접촉전압 저감:</b> 사람이 동시에 접촉할 수 있는 도전성부분 사이의 전위차를 낮춥니다.</li>
          <li><b>자동전원차단(ADS) 지원:</b> RCD·ELB 또는 과전류보호장치가 정해진 시간 안에 동작할 수 있도록 고장회로를 구성합니다.</li>
          <li><b>계통 기준전위 확보:</b> 변압기 중성점 등 전력계통의 기준점을 안정시킵니다.</li>
          <li><b>이상전압 처리:</b> LA·SPD가 제한한 낙뢰·개폐서지 전류를 접지계통으로 방류합니다.</li>
        </ul>
        <div class="formula-note"><b>원리 이해</b><span>U = I × Z</span><p>고장전류와 고장경로 임피던스에 의해 접촉전압이 발생합니다. 접지저항만 낮추는 것으로 끝나는 것이 아니라, 보호도체 연속성·등전위본딩·차단장치의 동작조건을 함께 만족해야 합니다.</p></div>
        <div class="warning-box"><b>오해 주의</b><p>“접지저항이 낮으면 무조건 안전하다”는 표현은 불완전합니다. 접지저항이 양호해도 PE 단선, 잘못된 N·PE 결합, 부적절한 RCD 감도 또는 차단시간 때문에 감전보호가 실패할 수 있습니다.</p></div>
        <p class="memory-line"><b>핵심:</b> 접지는 전위를 0V로 만드는 설비가 아니라, 위험한 전위차를 줄이고 보호장치가 신속히 동작하도록 만드는 보호시스템입니다.</p>
      </div></details>

      <details><summary>2. 계통접지와 TN·TT·IT 방식</summary><div class="training-body">
        <p><b>계통접지</b>는 전원계통의 한 점, 일반적으로 변압기 중성점을 대지와 연결하는 것입니다. 계통 구분은 접지극 개수보다 <b>전원측 접지상태와 노출도전성부분의 연결방법</b>으로 판단합니다.</p>
        <div class="system-grid">
          <div class="system-box"><h3>TT 계통</h3><ul><li>전원측 중성점은 전원측 접지극에 접지합니다.</li><li>수용가의 노출도전성부분은 수용가 자체 접지극과 PE에 연결합니다.</li><li>지락전류 귀로에 대지가 포함되어 고장전류가 작을 수 있으므로 RCD에 의한 보호가 핵심입니다.</li></ul><p class="memory-line"><b>쉽게:</b> 전원측 접지와 수용가 보호접지가 서로 독립된 구조입니다.</p></div>
          <div class="system-box"><h3>TN 계통</h3><ul><li>변압기 중성점을 접지합니다.</li><li>그 접지점과 수용가의 노출도전성부분을 PE 또는 PEN으로 금속적으로 연결합니다.</li><li>지락전류가 금속 귀로를 통해 변압기로 돌아가므로 큰 고장전류로 신속한 차단을 기대할 수 있습니다.</li></ul><p class="memory-line"><b>쉽게:</b> 전원에서부터 고장전류가 돌아갈 금속 경로가 함께 제공됩니다.</p></div>
          <div class="system-box"><h3>IT 계통</h3><ul><li>전원측을 대지와 절연하거나 높은 임피던스를 통해 접지합니다.</li><li>노출도전성부분은 접지합니다.</li><li>첫 번째 지락 시 운전을 유지할 수 있으나 절연감시장치(IMD)와 신속한 고장점 탐지가 필요합니다.</li></ul><p class="memory-line"><b>쉽게:</b> 첫 지락은 경보 후 유지할 수 있지만 두 번째 고장 전에 반드시 제거합니다.</p></div>
        </div>
        <div class="table-wrap"><table class="report-table"><thead><tr><th>구분</th><th>TT</th><th>TN</th><th>IT</th></tr></thead><tbody><tr><th>고장전류 귀로</th><td>대지 포함</td><td>PE/PEN 금속 귀로</td><td>첫 지락은 매우 제한적</td></tr><tr><th>주요 보호수단</th><td>RCD·등전위본딩</td><td>과전류보호장치·RCD</td><td>IMD·고장점 탐지</td></tr><tr><th>현장 판정 핵심</th><td>수용가 자체 접지극</td><td>전원 중성점과 PE의 금속 연결</td><td>전원 대지 절연 또는 임피던스 접지</td></tr></tbody></table></div>
        <p class="basis"><b>판정 순서:</b> 단선결선도 확인 → 변압기 중성점 접지 확인 → N·PE 결합 위치 확인 → PE/PEN의 공급 여부 확인 → 지락전류 귀로 확인.</p>
      </div></details>

      <details><summary>3. 주접지단자(MET)와 등전위본딩</summary><div class="training-body">
        <h3 class="training-title">■ MET의 역할</h3>
        <p>주접지단자(MET)는 보호도체, 접지도체, 주요 등전위본딩도체를 한 지점에서 연결하고 점검할 수 있도록 구성한 접지계통의 중심점입니다.</p>
        <ul><li>설비 외함, 철골, 케이블트레이, 피뢰·서지보호 접지 등 현장 설계에 따른 도체를 본딩합니다.</li><li>설비 사이의 전위차를 줄여 접촉전압을 낮춥니다.</li><li>점검 시 접지경로의 연속성과 접속상태를 체계적으로 확인할 수 있습니다.</li></ul>
        <div class="warning-box"><b>중요</b><p>MET에서 보호도체를 본딩하는 것과 부하측 분전반에서 N과 PE를 다시 연결하는 것은 다릅니다. 부하측 N·PE 재결합은 정상 중성선전류를 PE·철골·배관으로 분류시켜 누전차단기 오동작, 전식, 감전위험의 원인이 될 수 있습니다.</p></div>
        <h3 class="training-title">■ 현장 확인사항</h3>
        <ul><li>접지단자 풀림, 부식, 변색, 과열 및 도체 단선 여부</li><li>PE와 N의 식별 및 결합 위치</li><li>점퍼를 통한 공통접지망 구성과 측정 시 병렬경로 영향</li><li>피뢰설비·SPD 접지와 보호접지의 본딩상태</li></ul>
      </div></details>

      <details><summary>4. 접지저항 측정 목적과 2단자법</summary><div class="training-body">
        <h3 class="training-title">■ 측정 목적</h3>
        <ul><li>접지극과 접지도체의 열화·단선·접속불량 확인</li><li>과거 측정값과 비교한 저항 증가 추세 확인</li><li>접촉전압 저감과 보호장치 동작조건 검토를 위한 기초자료 확보</li></ul>
        <h3 class="training-title">■ 2단자법의 의미</h3>
        <p>2단자법은 측정 대상 접지와 이미 접지된 다른 도체를 이용해 형성된 <b>폐회로 전체의 합성저항</b>을 측정하는 간이방법입니다.</p>
        <div class="formula-note"><b>단순화한 관계</b><span>측정값 ≈ 대상 접지저항 + 보조 접지저항</span><p>병렬 접지경로가 없고 보조접지저항을 알고 있을 때에만 대상 접지저항을 근사할 수 있습니다.</p></div>
        <h3 class="training-title">■ 분전반에서 낮게 측정되는 이유</h3>
        <p>중성점 접지, 철골, 피뢰설비, 여러 접지극이 MET에서 공통으로 본딩되어 있으면 계측기는 개별 접지극이 아닌 전체 병렬망의 합성저항을 읽을 수 있습니다.</p>
        <div class="warning-box"><b>작업 주의</b><p>접지 점퍼나 중성점 접지를 임의로 분리하지 마십시오. 분리 측정이 필요하면 정전, 위험성평가, 계통검토와 작업절차를 먼저 수립해야 합니다.</p></div>
        <p class="memory-line"><b>핵심:</b> 2단자 측정값은 “어느 접지극의 값인지”보다 “측정 순간 어떤 경로가 폐회로를 구성했는지”를 먼저 해석해야 합니다.</p>
      </div></details>

      <details><summary>5. 절연저항 측정과 시험전압 선정</summary><div class="training-body">

        <ol class="step-list"><li>전원 차단, 무전압 확인, 잔류전하 방전</li><li>PLC, 인버터, 계측기, LED 드라이버, 통신기기, SPD·MOV 등 전자회로 연결 여부 확인</li><li>제조사 지침과 회로 정격에 따라 시험전압 선정 또는 민감기기 분리</li><li>측정 후 잔류전하 방전 및 결선 원상복구 확인</li></ol>

        <ul><li>전자회로나 서지보호소자가 연결된 회로는 DC 500V 인가로 손상될 수 있으므로 <b>DC 250V 적용 또는 민감기기 분리</b>를 검토합니다.</li><li>저압회로 절연저항의 적합기준은 회로의 정격전압과 시험조건에 따라 달라질 수 있으므로 “모든 설비가 무조건 1MΩ”이라고 단정하지 않습니다.</li><li>교육 메모의 1MΩ은 저압설비 현장판정에서 자주 사용하는 관리기준으로 활용하되, 최종 판정은 최신 KEC 표와 설비기준을 확인합니다.</li></ul>
        <div class="warning-box"><b>고장 방지</b><p>SPD·MOV의 표시전압 또는 최대연속사용전압과 절연저항계 시험전압은 같은 개념이 아닙니다. 단순히 “MOV가 470V이므로 500V에서 반드시 파손된다”고 단정하지 말고 제품 사양과 회로 연결상태를 확인하십시오.</p></div>

      <details><summary>7. 차단기 명판 읽기와 누전차단기 시험</summary><div class="training-body">
        <div class="table-wrap"><table class="report-table"><thead><tr><th>표시</th><th>의미</th><th>실무 확인</th></tr></thead><tbody><tr><th>AF</th><td>프레임 정격</td><td>차단기 몸체가 수용할 수 있는 최대 등급</td></tr><tr><th>AT / A</th><td>트립 정격·정격전류</td><td>부하전류, 전선 허용전류와의 보호협조</td></tr><tr><th>kA</th><td>정격차단전류</td><td>설치점 예상단락전류 이상인지 확인</td></tr><tr><th>V / kV</th><td>정격전압</td><td>계통전압 이상 및 극수·주파수 확인</td></tr><tr><th>IΔn</th><td>정격감도전류</td><td>감전보호·화재보호 목적에 맞는 감도 선정</td></tr></tbody></table></div>
        <h3 class="training-title">■ 30mA 누전차단기 해석</h3>
        <ul><li>정격감도전류가 30mA인 제품은 규정된 시험조건에서 30mA 이하에서 동작해야 합니다.</li><li>정격부동작전류는 일반적으로 정격감도전류의 1/2인 15mA이며, 이 값 이하에서는 동작하지 않아야 합니다.</li><li>실제 동작전류는 15mA 초과 30mA 이하 범위에 있을 수 있으므로 “항상 24mA에서 동작”한다고 고정하지 않습니다.</li><li>0.03초 이내 표시는 고속형 제품의 동작시간 등급이며 실제 시험은 전용 시험기로 동작전류와 동작시간을 각각 확인합니다.</li></ul>
        <h3 class="training-title">■ 인체 영향</h3>
        <p>감전위험은 전류 크기뿐 아니라 통전시간, 통전경로, 주파수, 피부상태에 따라 달라집니다. 수십 mA의 교류전류도 심실세동 등 치명적 결과를 일으킬 수 있으므로 보호장치의 신속한 차단이 중요합니다.</p>
        <div class="warning-box"><b>시험 주의</b><p>TEST 버튼은 기계적·전기적 동작 여부를 확인하는 일상점검 수단이며, 정격감도전류와 동작시간의 정량시험을 대신하지 않습니다.</p></div>
      </div></details>

      <details><summary>8. SPD·MOV·LA·SA의 역할과 설치원칙</summary><div class="training-body">
        <h3 class="training-title">■ 장치 구분</h3>
        <ul><li><b>SPD:</b> 저압설비에서 낙뢰·개폐서지에 의한 과도과전압을 제한하는 장치</li><li><b>MOV:</b> SPD 내부에 널리 사용되는 비선형 저항소자</li><li><b>LA:</b> 고압·특고압 계통에서 서지를 제한하여 변압기와 기기를 보호하는 피뢰기</li><li><b>SA:</b> 계통과 제조사 용어에 따라 서지흡수기·서지보호장치를 의미하므로 도면과 사양서를 확인</li></ul>
        <h3 class="training-title">■ 병렬·직렬 설치</h3>
        <ul><li>SPD·LA처럼 <b>전압을 제한하는 장치</b>는 보호대상과 병렬로 접속합니다.</li><li>VCB·ACB·MCCB·ELB처럼 <b>회로전류를 개폐·차단하는 장치</b>는 회로에 직렬로 설치합니다.</li></ul>
        <h3 class="training-title">■ 22.9kV 계통의 LA 이해</h3>
        <p>22.9kV는 선간전압이며, 중성점이 접지된 3상 계통의 정상 상전압은 약 <b>22.9/√3 = 13.2kV</b>입니다. 18kV급 LA의 정격은 단순한 “18kV 동작스위치”가 아니라, 정상 상전압과 일시적 과전압을 견디면서 정해진 제한전압·방전특성을 확보하도록 선정된 등급입니다.</p>
        <h3 class="training-title">■ 현장 점검</h3>
        <ul><li>SPD 상태표시창, 분리기, 백업보호장치와 접속단자 확인</li><li>접지선은 가능한 짧고 직선으로 시공하고 불필요한 루프와 급격한 굴곡을 피함</li><li>LA 접지선, 단자, 애자 오염·균열·누설흔적 및 접지연속성 확인</li></ul>
        <p class="memory-line"><b>기억:</b> 전압 제한 장치는 병렬, 전류 차단 장치는 직렬입니다.</p>
      </div></details>

      <details><summary>11. 현장 오판 방지 체크리스트</summary><div class="training-body">
        <ul class="check-list">
          <li>접지저항 수치만으로 접지계통의 안전성을 단정하지 않는다.</li>
          <li>TT·TN은 접지극 개수가 아니라 N·PE 연결과 고장전류 귀로로 구분한다.</li>
          <li>부하측에서 N과 PE를 임의로 연결하지 않는다.</li>
          <li>2단자 접지저항 측정값을 개별 접지극 저항으로 바로 해석하지 않는다.</li>

    (function () {
      function showUpdateNotice(registration) {
        if (document.getElementById('pwaUpdateNotice')) return;
        var box = document.createElement('div');
        box.id = 'pwaUpdateNotice';
        box.className = 'pwa-update-notice';
        box.innerHTML = '<span>새 버전이 있습니다.</span><button type="button">업데이트</button>';
        document.body.appendChild(box);
        box.querySelector('button').addEventListener('click', function () {
          if (registration && registration.waiting) {
            registration.waiting.postMessage({ type: 'SKIP_WAITING' });
          } else {
            window.location.reload();
          }
        });
      }

      if ('serviceWorker' in navigator) {
        var refreshing = false;
        navigator.serviceWorker.addEventListener('controllerchange', function () {
          if (refreshing) return;
          refreshing = true;
          window.location.reload();
        });

        window.addEventListener('load', function () {
          navigator.serviceWorker.register('./service-worker.js').then(function (registration) {
            if (registration.waiting && navigator.serviceWorker.controller) {
              showUpdateNotice(registration);
            }
            registration.addEventListener('updatefound', function () {
              var newWorker = registration.installing;
              if (!newWorker) return;
              newWorker.addEventListener('statechange', function () {
                if (newWorker.state === 'installed' && navigator.serviceWorker.controller) {
                  showUpdateNotice(registration);
                }
              });
            });
          }).catch(function (err) {
            console.warn('Service worker registration failed:', err);
          });
        });
      }
    })();
