      <details open><summary>1. 접지의 목적과 현장 판단</summary><div class="training-body">
        <h3 class="training-title">■ 접지를 설치하는 이유</h3>
        <ul>
          <li><b>감전 예방:</b> 누전이나 지락으로 외함에 전압이 발생했을 때 사람이 접촉하는 전압을 낮춥니다.</li>
          <li><b>보호장치 동작 확보:</b> 고장전류가 되돌아갈 경로를 만들어 ELB·RCD 또는 과전류차단기가 동작할 수 있도록 합니다.</li>
          <li><b>전위상승 억제:</b> 접지저항을 낮춰 고장 시 외함과 접지망의 전위상승을 줄입니다.</li>
          <li><b>기준전위 확보:</b> 변압기 중성점과 전력계통의 전압 기준점을 안정시킵니다.</li>
          <li><b>이상전압 방전:</b> LA·SPD가 제한한 낙뢰·개폐서지를 접지로 방전합니다.</li>
        </ul>
        <div class="formula-note"><b>이해용 관계식</b><span>V = I × R</span><p>같은 고장전류가 흐른다면 접지저항이 낮을수록 전위상승과 접촉전압이 작아집니다. 계산 자체보다 “접지저항을 낮게 유지하는 이유”를 이해하기 위한 식입니다.</p></div>
        <h3 class="training-title">■ 실무에서 함께 확인할 사항</h3>
        <ul>
          <li>측정값뿐 아니라 이전 측정값과 비교하여 증가 추세를 확인합니다.</li>
          <li>접지선 단선, 단자 풀림, 부식, 접지단자함의 본딩 상태를 함께 점검합니다.</li>
          <li>접지가 양호해도 보호장치의 종류와 동작조건이 맞지 않으면 자동 차단이 보장되지 않습니다.</li>
        </ul>
        <p class="memory-line"><b>한 줄 정리:</b> 접지는 단순히 저항값을 낮추는 작업이 아니라, 사람의 접촉전압을 낮추고 보호장치가 동작할 수 있는 고장전류 경로를 확보하는 설비입니다.</p>
      </div></details>

      <details><summary>2. 계통접지와 TN·TT·IT 방식</summary><div class="training-body">
        <p><b>계통접지</b>는 변압기 중성점 등 전력계통의 한 점을 대지와 연결하여 전압 기준점을 만들고, 고·저압 혼촉이나 지락 시 보호장치가 동작하도록 하는 접지입니다.</p>
        <div class="system-grid">
          <div class="system-box"><h3>TT 방식</h3><ul><li>전원측 중성점은 전원측에서 접지합니다.</li><li>수용가 설비 외함은 수용가가 설치한 자체 접지극과 PE에 연결합니다.</li><li>고장전류가 대지를 포함한 경로로 귀환하므로 전류가 작을 수 있어 RCD·ELB의 역할이 중요합니다.</li></ul><p class="memory-line"><b>쉽게:</b> 전원측은 N의 기준점을 만들고, 수용가는 자체 PE 접지망을 구축합니다.</p></div>
          <div class="system-box"><h3>TN 방식</h3><ul><li>변압기 중성점을 접지합니다.</li><li>그 접지점과 수용가 외함을 PE 또는 PEN 도체로 금속적으로 연결합니다.</li><li>지락전류가 PE·PEN을 통해 변압기 중성점으로 되돌아가므로 큰 고장전류로 신속한 차단을 기대할 수 있습니다.</li></ul><p class="memory-line"><b>쉽게:</b> 변압기에서부터 고장전류의 금속 귀로가 함께 공급됩니다.</p></div>
          <div class="system-box"><h3>IT 방식</h3><ul><li>전원측을 대지와 절연하거나 높은 임피던스를 통해 접지합니다.</li><li>설비 외함은 접지합니다.</li><li>첫 번째 지락 때 즉시 정전하지 않고 운전을 유지할 수 있지만 절연감시장치와 고장점 탐지가 필요합니다.</li></ul><p class="memory-line"><b>쉽게:</b> 첫 지락은 감시하면서 운전하고, 두 번째 고장 전에 반드시 조치합니다.</p></div>
        </div>
        <div class="table-wrap"><table class="report-table"><thead><tr><th>구분</th><th>TT</th><th>TN</th><th>IT</th></tr></thead><tbody><tr><th>보호접지 경로</th><td>수용가 자체 접지극</td><td>PE/PEN 금속 귀로</td><td>외함 접지극</td></tr><tr><th>지락전류 특성</th><td>비교적 작을 수 있음</td><td>비교적 크게 흐름</td><td>첫 지락은 작음</td></tr><tr><th>중점 보호수단</th><td>RCD·ELB</td><td>과전류차단기·RCD</td><td>절연감시·고장점 탐지</td></tr></tbody></table></div>
        <p class="basis"><b>판정 핵심:</b> 접지극의 개수나 현장 명칭만으로 TT·TN을 판단하지 않습니다. N과 PE가 어디에서 연결되고, 지락전류가 어떤 도체를 통해 변압기로 되돌아가는지를 확인해야 합니다.</p>
      </div></details>

      <details><summary>3. 기흥 사업소 접지시스템과 등전위본딩</summary><div class="training-body">
        <h3 class="training-title">■ 현장 구성</h3>
        <p>기흥 사업소는 현장 관리 기준상 <b>TT 접지방식</b>으로 운용하며, 변압기 중성점은 전원측 접지와 연결하고 수용가 내부의 보호접지는 자체 접지망으로 구성합니다.</p>
        <p>주접지단자함(MET)에는 설비 외함, 피뢰기(LA), SPD, 철골, 계장접지 등 보호 목적의 접지도체를 모아 <b>등전위본딩</b>합니다.</p>
        <h3 class="training-title">■ 등전위본딩의 실무 의미</h3>
        <ul>
          <li>설비마다 서로 다른 전위가 발생하는 것을 줄여 접촉전압과 설비 간 전위차를 낮춥니다.</li>
          <li>접지경로를 한곳에서 확인할 수 있어 점검과 측정이 쉬워집니다.</li>
          <li>여러 접지극이 공통망으로 연결되면 합성 접지저항이 낮아질 수 있습니다.</li>
          <li>공통접지망에서 고장전류 귀로의 임피던스가 낮아지면 고장전류가 증가하여 보호장치 동작에 유리할 수 있습니다.</li>
        </ul>
        <div class="warning-box"><b>주의</b><p>MET에서 보호접지를 본딩하는 것과 부하측 분전반에서 N선과 PE선을 임의로 다시 연결하는 것은 전혀 다른 작업입니다. 부하측 N·PE 재결합은 정상전류가 접지계통으로 흐르는 원인이 될 수 있습니다.</p></div>
        <p class="memory-line"><b>현장 확인 순서:</b> 단선결선도 → 변압기 중성점 접지 → MET 연결상태 → N·PE 결합 위치 → 지락전류 귀로.</p>
      </div></details>

      <details><summary>4. 접지저항 측정 목적과 2단자법</summary><div class="training-body">
        <h3 class="training-title">■ 측정 목적</h3>
        <ul><li>접지저항 변화와 접지극 열화 확인</li><li>접지도체 단선, 단자 풀림, 접속부 부식 확인</li><li>접촉전압 억제와 지락보호 시스템의 동작 가능성 확인</li></ul>
        <h3 class="training-title">■ 2단자법의 원리</h3>
        <p>2단자법은 측정 대상 접지와 이미 접지된 보조점을 이용하여 <b>폐회로의 합성저항</b>을 측정하는 간이 방법입니다.</p>
        <div class="formula-note"><b>이해용 관계</b><span>측정값 ≈ 주접지저항 + 보조접지저항</span><p>다른 병렬경로가 없고 보조접지저항을 알고 있을 때에만 주접지저항을 근사할 수 있습니다.</p></div>
        <h3 class="training-title">■ 분전반 측정 절차</h3>
        <ol class="step-list"><li>회로와 설비 상태를 확인하고 측정 중 위험이 없는지 판단합니다.</li><li>분전반의 중성선과 접지단자 중 측정 목적에 맞는 지점을 확인합니다.</li><li>접지저항계의 P·C 공통 리드와 E 리드를 제조사 방법에 따라 연결합니다.</li><li>측정 후 보조접지의 영향과 병렬 접지경로 유무를 함께 해석합니다.</li></ol>
        <h3 class="training-title">■ 공통접지망에서 낮게 측정되는 이유</h3>
        <p>접지단자함의 점퍼와 본딩을 분리하지 않으면 여러 접지극, 철골, 중성점 접지 등이 병렬로 연결된 전체 합성저항을 측정하게 됩니다. 따라서 개별 제3종 접지극의 실제 저항보다 낮은 값이 나올 수 있습니다.</p>
        <div class="warning-box"><b>측정 주의</b><p>접지단자함의 분리 작업은 반드시 정전 및 안전성 검토 후 시행합니다. 운전 중 임의로 중성점 접지나 보호접지를 분리하면 감전·설비손상 위험이 있습니다.</p></div>
        <p class="memory-line"><b>한 줄 정리:</b> 2단자법의 숫자는 항상 개별 접지극 값이 아니라, 측정 시 형성된 전체 귀로의 합성저항일 수 있습니다.</p>
      </div></details>

        <p class="memory-line"><b>기억하기:</b> 전압 제한 장치는 병렬, 전류 차단 장치는 직렬입니다.</p>

    (function () {
      function showUpdateNotice(registration) {
        if (document.getElementById('pwaUpdateNotice')) return;
        var box = document.createElement('div');
        box.id = 'pwaUpdateNotice';
        box.className = 'pwa-update-notice';
        box.innerHTML = '<span>새 버전이 있습니다.</span><button type="button">업데이트</button>';
        document.body.appendChild(box);
        box.querySelector('button').addEventListener('click', function () {
          if (registration && registration.waiting) {
            registration.waiting.postMessage({ type: 'SKIP_WAITING' });
          } else {
            window.location.reload();
          }
        });
      }

      if ('serviceWorker' in navigator) {
        var refreshing = false;
        navigator.serviceWorker.addEventListener('controllerchange', function () {
          if (refreshing) return;
          refreshing = true;
          window.location.reload();
        });

        window.addEventListener('load', function () {
          navigator.serviceWorker.register('./service-worker.js').then(function (registration) {
            if (registration.waiting && navigator.serviceWorker.controller) {
              showUpdateNotice(registration);
            }
            registration.addEventListener('updatefound', function () {
              var newWorker = registration.installing;
              if (!newWorker) return;
              newWorker.addEventListener('statechange', function () {
                if (newWorker.state === 'installed' && navigator.serviceWorker.controller) {
                  showUpdateNotice(registration);
                }
              });
            });
          }).catch(function (err) {
            console.warn('Service worker registration failed:', err);
          });
        });
      }
    })();
  </script>
